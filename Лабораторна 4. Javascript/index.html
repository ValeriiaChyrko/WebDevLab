Завдання №1. Надайте відповідь, який скрипт виконається першим? Аргументуйте відповідь одним реченням.

Першим виконається скрипт - big.js.
Якщо у файлі з’являється більше одного скрипта, то вони виконуються в тому порядку, в якому вони з’являються в коді.

Завдання №4. Надати відповідь, чому дорівнюватиме х у цьому прикладі:
var a = 2;
var x = 1 + (a *= 2).

Розглянемо порядок виконання компілятором виразу: x = 1 + (a *= 2)
Першим виконається арифметична операція множення із присвоєнням:
а *= 2 -> а = 2*2 -> a = 4.
Далі до результату виконання попередньої операції додається 1:
x = 4+1 -> x=5.

Завдання №5. Відповісти на питання, чи видається alert у такому випадку:
if ("0") {
alert( ‘Привіт’ );
}

Так, alert виведеться на екран у зв'язку з тим, що перевіряється значення рядка "0". Відповідно до документації будь-який непорожній рядок є true, тоді як порожній - false.
